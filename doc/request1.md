# Tom’s Advanture

## 游戏简介：

玩家通过打小怪的方法通过一个个关卡。玩家可以通过键盘上的上下左右键进行走位控制，通过鼠标进行战斗，左键按下表示攻击，如果当前的武器是枪，左键按下位置和主角位置的相对方向就是射击的方向，如果当前武器是砍刀，则左键按下对较小范围内的敌人造成伤害（称为普攻）。武器的切换通过鼠标选择进行。游戏的机制是顶视射击，采用手动瞄准的设计，清理完一个房间的小怪就可以去下一个房间，找到传送门打败大Boss，即可进入下一个关卡。玩家有一个血条记录当前血量，被小怪攻击到就会降低血量，血条为0，游戏结束。

## 基本功能：

1. 背景是以主角为中心的俯视图，会随着主角的移动而变化，始终以主角为中心。
2. 按下键盘上的方向键，主角能够移动。
3. 射击时，人物图像能朝左键按下位置发射子弹。
4. 被小怪的攻击打中，主角掉血，血条清0后游戏结束
5. 小怪会向主角的方向移动并攻击，但攻击不是自瞄准的，而是朝移动方向发射子弹。
6. 小怪有一定的血量，被主角攻击中会掉血，血条为0，小怪死亡并消失。
7. 右上角有暂停键，左键点击后可以暂停游戏，或者按空格键暂停游戏。
8. 右上角有退出键，左键点击或者按“ESC”键可以退出游戏，在此之前，对游戏存档，保存游戏进度，在下次进入时自动加载。
9. 角色死亡后回到主界面，可以选择“新游戏”开始新一局，也可以选择“继续游戏”，在死亡关卡的开始房间重新开始游戏。

## 拓展功能：

1. 增加子弹数，主角每射击一次，子弹数减一，子弹数为0，玩家只能选择普攻。
2. 打死小怪可获得金币，通过金币能够购买药剂，增加生命值。
3. 在主角没有杀光房间内所有小怪是，通道不开启，无法进入下一个房间。
4. 主角进入传送门，切换到下一个关卡。



## 第一轮迭代计划：
  设计登陆界面，页面上显示“新游戏”、“继续游戏”和“关闭”按钮，点击“新游戏”即可进入游戏；点击“继续游戏”可以回到存档的位置（如果上次的游戏进度以死亡告终，则游戏存档点在该关卡的开始房间）；点击“关闭”按钮，关闭整个界面。
